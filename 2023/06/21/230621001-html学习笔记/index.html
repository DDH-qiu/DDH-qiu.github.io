html学习笔记

2023-06-21 学习笔记 web 评论 字数统计: 1.6k(字) 阅读时长: 6(分)

html学习笔记

1-标签的关系

父子关系（嵌套关系）
兄弟关系（并列关系）

2-注释

对代码进行解释说明工作

vscode中使用快捷键ctrl+/

3-标题标签

标签名：h1-h6（双标签）
h1-h6逐级递减
h1-标题/logo
独占一行

4-段落标签

P标签（双标签）
文章段落，描述信息
特点：独占一行，段落之间有间隙，会随着界面大小换行

5-换行标签

<br>单标签
不识别回车键使用<br>进行换行

6-水平线效果

<hr>单标签，增加水平线的效果

7-文本格式化标签

行内元素
开始标签 内容 结束标签
作用：为文本添加特殊格式，以突出文本
加粗 strong b
倾斜 em i
下划线 ins u
删除线 del s
strong，em，ins，del自带强调含义（常用）

8-图像标签

单标签，行内元素
<img src="URL">
src指定图片位置
图像标签属性
alt - 替代文本
title - 提示文本
width - 图片的宽度
height - 图片的高度
不区分先后顺序

9-网页资源路径

相对路径
绝对路径
网址链接路径
当前文件夹 ./
上一级文件夹 ../

10-超链接

<a href="URL"></a>
URL-网址，页面地址等
超链接属性
target="_blank"新窗口打开页面

11-音频标签

<audio src="URL"><audio>
URL-音频地址
音频标签常用属性
src 地址（必有） MP3，OGG，WAV格式
controls 显示控制面板
loop 循环播放
dutoplay 自动播放 浏览器默认不播放
（html5中，属性与值相同可只写一个（简写化））

12-视频标签

<video src="URL"><video>
src 视频地址 MP4，webm，ogg
controls 显示视频控制面板
loop 循环播放
mvted 静音播放
autoplay 自动播放 浏览器默认不播放

13-列表

布局内容排列整齐的区域
分类：无序列表，有序列表，定义列表

13.1-无序列表

布局排列整齐的不需要规定顺序的区域
标签：ul嵌套li，ul是无序列表，li是列表条目
ul标签里只包含li标签
li标签里可以包裹任何内容

13.2有序列表

布局排列整齐的需要规定顺序的区域
标签：ol嵌套li，ol是有序列表，li是列表条目
ol标签里只包含li标签
li标签里可以包裹任何内容

13.3定义列表

dl嵌套dt和dd，dl是定义列表，dt是定义列表的标题，dd是定义列表的描述/详情
dl里面只能包含dt和dd标签
dt和dd里面可以包含任何内容

14-表格

用来展示数据，双标签
标签：table嵌套tr，tr嵌套td/th
table 表格
tr 行
th 表头单元格
td 内容单元格
在网页中，表格默认没有框线，使用border属性可以为表格添加边框线
<table border="像素大小"></table>

14.1表格结构标签

用表格结构标签把内容划分区域，让表格结构更清晰，语义更清晰
thead 表格头部
tbody 表格主体
tfoot 表格底部

14.2合并单元格

将多个单元格合并成一个单元格，以合并同类信息
·跨行合并
·跨列合并
合并单元格的一般步骤
（1）明确合并的目标
（2）保留最左最上的表格，添加属性，取值是数字，表示需要合并的单元格数量
1）跨行合并：保留最上单元格，添加属性rowspan
2）跨列合并：保留最左单元格，添加属性colspan
（3）删除其他单元格

15-表单

收集用户信息（登录页面，注册页面，搜索页面）

15.1input标签基本使用

文本框，密码框
<input type="...">
type属性值：
text 文本框（单行文本）
password 密码框 ***
radio 单选框
checkbox 多选框
file 上传文件

15.2input标签占位文本

占位文本：提示信息
<input type="..." blaceholder="提示信息">
文本框密码框均可用

15.3单选框

常用属性
name 控件名称 控件分组，同组只能选中一个
checked 默认选中值 属性名和属性值相同，简写为一个单词

15.4上传文件

默认只能上传一个文件，添加multiple属性可以实现多个文件上传功能
<input type="file" multiple>

15.5多选框（复选框）

默认选中：checked
<input type="checkbox" checked>

16-下拉菜单

单击下拉事件
标签：select嵌套option
select是下拉菜单整体
option是下拉菜单的每一项
selected属性默认选中

17-文本域

多行输入文本的表单控件
标签：textared 双标签
<textarea>提示文字</textarea>
拖拽放大缩小功能一般禁用

18-label标签

为标签添加说明文本
可用label标签绑定文字和表单控件的关系，增大表单控件的点击范围
写法1：
·label标签只包裹内容，不包裹表单控件
·设置label标签的for属性值和表单控件的id属性值相同
<input type="radio" id="man"><label for="man">男<label>
写法2：
使用label标签包裹文字和表单控件不需要属性
<babel><input type="radio">女</label>
适用范围：文本框，密码框，上传文件，单选框，多选框，下拉菜单，文本域等

19-按钮

button标签
<button type="">按钮</button>
type属性值：
submit 提交按钮 点击可将数据提交到后台
reset 重置按钮 将表单控件恢复默认值
button 普通按钮 没有功能，配合JavaScript使用
*<form action="发送数据地址"></from>
使用from标签统一管理

20-无语义的布局标签

作用：布局网页，划分网页区域，摆放内容
div 独占一行
span 不换行

21-字符实体

在网页中显示预留字符
空格 space &nbsp;
小于号 < <
大于号 > >
